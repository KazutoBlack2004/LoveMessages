## ✨ Características Principales

### ✉️ Sobre Interactivo (Interactive Envelope)
Un sobre físico digitalizado que se abre para revelar una carta personalizada.
- Animación suave de apertura con **Framer Motion**.
- Z-index dinámico para una inmersión completa.
- Partículas de corazones animados al abrirse.

### 💍 Propuesta Directa (Proposal Experience)
La forma definitiva de hacer "la pregunta".
- **Botón "No" Inteligente**: Un botón travieso que escapa del cursor, haciendo que la única opción real sea el "Sí".
- Transiciones fluidas y diseño minimalista enfocado en la respuesta.

### 🍱 Plantillas Temáticas
Diseños curados para cada ocasión:
- **Carta de Amor**: Estética romántica con tipografía serif y tonos rosados.
- **Feliz Cumpleaños**: Vibrante y festivo con iconos de pasteles y confeti.
- **Modo Oscuro Premium**: Cada plantilla cuenta con su versión dark-mode optimizada con efectos de "Cristal Tintado" (Glassmorphism) de alto nivel.

## 🛠️ Stack Tecnológico

- **Frontend**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Estilos**: [Tailwind CSS 4.0](https://tailwindcss.com/) (Custom Design Tokens)
- **Animaciones**: [Framer Motion](https://www.framer.com/motion/)
- **Iconografía**: [Lucide React](https://lucide.dev/)
- **Backend (Opcional/Configurable)**: Integración con [Supabase](https://supabase.com/) para almacenamiento de mensajes.

## 📦 Instalación y Uso Local

1. Clona el repositorio:
```bash
git clone https://github.com/KazutoBlack2004/LoveMessages.git
```

2. Instala las dependencias:
```bash
npm install
```

3. Ejecuta el servidor de desarrollo:
```bash
npm run dev
```

---

## 🎨 Diseño y Estética

El proyecto sigue una filosofía de **Arquitectura de Diseño Premium**:
- **Paleta Armoniosa**: Uso de colores `primary` (rosas) y `gray` (mauves) personalizados en `index.css`.
- **Efecto Cristal**: Implementación de `backdrop-blur` extremo y transparencias tintadas para una interfaz que se siente viva.
- **Tipografía**: Combinación de *Inter* para interfaz y *Playfair Display* para mensajes emotivos.
